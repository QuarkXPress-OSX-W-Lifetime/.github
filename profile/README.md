## Overview

QuarkXPress for macOS is a specialized desktop publishing solution designed for users who want a reliable, OSX-native tool with clear controls and stable performance.

The native integration works seamlessly with macOS services and features. Launch through Spotlight, preview with Quick Look, or access through Services menu.

---

## Key Features

- Native macOS application
- Universal binary for all Macs
- Dark mode support
- Quick Look integration
- Spotlight indexing
